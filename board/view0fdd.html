오산양조 #하얀까마귀

​

✔️제품명 : 하얀까마귀(나이스 에디션)

✔️양조장 : 농업회사법인 오산양조(주)

✔️종류 : 탁주

✔️용량 : 500ml

✔️도수 : 8%

✔️보관온도 : 10℃ 이하 냉장보관

✔️병입일 : 2023.04.03

✔️유통기한 : 2023.05.03.(병입일로부터 30일)

✔️원재료명 : 정제수, 쌀(국내산), 국[입국, 곡자(밀), 조효소제, 정제효소], 효모

✔️감미료 : 무

✔️살균 : 무

✔️가격 : ￦20,000/3병(술담화)

✔️브랜딩(키워드) : 쌀알 모양의 하얀 까마귀

✔️패키징(병) : 높이가 낮고 원기둥 모양의 페트병

✔️패키징(라벨링) : 나이스 에디션으로 하얀까마귀가 골프채를 들고 있는 모습

✔️ETC : 마을 기업, 찾아가는 양조장, 삼양주 방식, 진주곡자 사용.

✔️색상 : 우윳빛에 가까운 야쿠르트색

✔️향 : 우유, 요거트, 꿀

✔️맛 : 무게감이 있지만 부드럽고 달달함. 쫀득함. 담백함. 깔끔함. 맛이 단조로움.

#에디_테이스팅노트

첫 맛에 우유와 같은 밀키함과 꿀과 같은 달달함이 느껴지면서 무게감이 살짝 있지만 목넘김이 부드럽고 여운이 길지 않게 깔끔하다. 다만 마신 후 혀에 남겨지는 미세한 쌀 알갱이들이 느껴진다. 이 알갱이들에 대해 궁금하여 오산양조에 질문을 했었는데, 하얀까마귀가 수작업이 아닌 기계식 공정으로 진행 중 제성 때, 발효로 삭은 쌀알이 제성기를 통과하면서 고형분과 액체로 분리되는데, 이 때 쌀로 인한 고형분이 술에 포함된다고 보면 된다고 오산 양조측 답변을 받았으며, 단순히 쌀알갱이라고 생각하면 된다. 전체적인 맛은 단조로우면서 담백하고 깔끔한데, 맛으로 보면 최근 젊은 층을 타겟으로 하여 순한 단맛, 우유 같은 밀키함, 강하지 않은 알콜감, 목넘김이 깔끔하지만 무게감이 있어 쫀득감을 주어 폭 넓은 연령층이 즐길 수 있을 거라 생각한다. 전체적으로 담백한 탁주로 자극적이지 않은 두부김치, 보쌈 등과 일반적인 식사의 반주로도 페어링이 좋다고 생각된다.

진주곡자를 사용함에 따라 유통기한에 다다르면서 당도는 줄고 좀 더 깊은 맛을 낸다는 오산양조 김유훈 대표님의 말씀을 직접 듣고, 남은 하얀까마귀는 유통기한이 다다를 때쯤 다시 한 번 후기를 쓰도록 하겠습니다.
